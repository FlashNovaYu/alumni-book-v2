# 同学录导航栏重构设计说明书 (Navigation Bar Redesign Spec)

## 1. 设计目标与背景
现有导航栏（TopNav）已经具备了“书本”、“印章”等基础的模拟纸质（Analog）风格。本次重构的目标是在保持现有骨架布局的前提下，将其视觉质感提升至“典藏档案级”（Premium Analog），赋予其更强的高级感、艺术化表现和深邃的视觉层次。

## 2. 核心视觉风格 (Premium Analog)
- **色调调整**：将主色调向更加沉稳、古典的方向靠拢。使用深栗色、复古墨色作为文字主色，背景引入极其细微的亚麻/羊皮纸纹理或温润的米黄色底色。
- **排版 (Typography)**：提升文字的排版艺术感，将导航链接字体调整为优雅的衬线体（Serif）或进行加粗、适当增加字间距（Letter-spacing），使其具有老式印刷品的质感。
- **装饰元素**：用“烫金”、“铜板雕刻”或“凹凸压花”的质感取代过于明亮的红色印章动画。

## 3. 布局与结构 (B1 经典三段式)
保持现有的三段式灵活布局，不改变用户原有的操作习惯：
- **左侧 (Brand)**：系统 Logo 与“同学录”文字。字体字号优化，更加醒目且富有品牌感。
- **中部 (Directory)**：导航菜单项（前言、同学档案、班级空间、年度册、更多）。
- **右侧 (Utilities)**：主题切换、管理后台入口、登录状态与信箱。

## 4. 交互与微动效 (Micro-animations)
- **悬停效果 (Hover)**：鼠标悬停在导航项时，触发轻微的“压印 (Letterpress)”或“烫金发光”效果，而非简单的颜色突变。
- **选中状态 (Active)**：底部的指示器（原本的 `nav-active-ink` 和 `nav-active-paper`）将重写为更高级的指示样式（如古典的下划双线或细致的边框纹理）。
- **滚动状态 (Scrolled)**：页面向下滚动时，导航栏底部的阴影会变得更深邃，仿佛一本厚重的书页悬浮在内容之上。

## 5. 影响的代码文件
1. `packages/site-astro/src/components/TopNav.astro`：修改 HTML 结构与核心 CSS 样式。
2. `packages/site-astro/src/components/TopNavSession.vue`：同步调整 Vue 客户端交互岛的样式，确保与 Astro 静态部分风格完全一致。

## 6. 后续实施步骤
在通过用户评审后，将使用 `writing-plans` 技能制定包含代码重构、CSS 变量替换、测试的逐步实施计划。
